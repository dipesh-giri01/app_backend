# Nepali Chess Players API

## Data Field Legend

| Field           | Description                                                                                       |
|-----------------|---------------------------------------------------------------------------------------------------|
| `STD`/`SRTNG`   | Standard rating                                                                                   |
| `RPD`/`RRTNG`   | Rapid rating                                                                                      |
| `BLZ`/`BRTNG`   | Blitz rating                                                                                      |
| `SGM`           | Number of STANDARD rated games in given period                                                    |
| `RGM`           | Number of RAPID rated games in given period                                                       |
| `BGM`           | Number of BLITZ rated games in given period                                                       |
| `SK`            | STANDARD rating K factor (2-digit)                                                                |
| `RK`            | RAPID rating K factor (2-digit)                                                                   |
| `BK`            | BLITZ rating K factor (2-digit)                                                                   |
| `B-day`/`BORN`  | Year of birth of a player                                                                         |
| `ID NUMBER`     | Identification number of a player within FIDE database                                            |
| `NAME`          | Name of a player                                                                                  |
| `TIT`/`TITL`    | Title of a player (GM - Grand Master, WGM - Woman Grand Master, IM - International Master, etc.)  |
| `FED`           | Federation of a player (e.g., NEP for Nepal)                                                      |
| `OTIT`          | Other titles (IA - International Arbiter, FA - FIDE Arbiter, NA - National Arbiter, etc.)         |
| `FLAG`          | Flag of inactivity (I - inactive, WI - woman inactive, w - woman)                                 |
| `SEX`           | Sex of a player (M - male, F - female)                                                            |

### Title Abbreviations
- `GM`: Grand Master
- `WGM`: Woman Grand Master
- `IM`: International Master
- `WIM`: Woman International Master
- `FM`: FIDE Master
- `WFM`: Woman FIDE Master
- `CM`: Candidate Master
- `WCM`: Woman Candidate Master

### Other Titles
- `IA`: International Arbiter
- `FA`: FIDE Arbiter
- `NA`: National Arbiter
- `IO`: International Organizer
- `FT`: FIDE Trainer
- `FST`: FIDE Senior Trainer
- `DI`: Developmental Instructor
- `NI`: National Instructor
